## leet语介绍

Leet，又称黑客语，是指一种发源于欧美地区的BBS、线上游戏和黑客社群所使用的文字书写方式。通常是把[拉丁字母](https://baike.baidu.com/item/拉丁字母/1936851?fromModule=lemma_inlink)转变成数字或是特殊符号，例如E写成3、A写成@等。或是将单字写成同音的字母或数字，如to写成2、for写成4等等。Leet一词源于英语的elite，由elite转为eleet后再转变为leet。

## 工具介绍

通过输入字符串，并随机替换其中的字符，以实现leet语的随机化生成

字典包含在代码中，修改leet_list即可，也可自由添加
